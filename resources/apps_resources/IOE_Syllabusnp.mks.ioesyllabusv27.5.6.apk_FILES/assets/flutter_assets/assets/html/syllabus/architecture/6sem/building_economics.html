Building Economics


Course Objectives:
To acquaint the students with some of the basic concepts and tools used to carry economic and financial analysis of building/engineering projects. To provide students with a more interdisciplinary approach to project planning, management and evaluation techniques.

Course topics:
  1. Economic Principle[8 hours]
    1. Economic equilibrium, demand, supply, production
    2. Welfare economics

  2. Planning the study[4 hours]
    1. Identifying goals and objects
    2. Organizing the study
    3. Administering the study
    4. Planning the budget

  3. Implementation/Evaluation Techniques[14 hours]
    1. Linear programming
    2. Cost/Benefit Analysis
    3. Direct/Indirect cost
    4. Tangible/Intangible benefit
    5. Break even analysis
    6. Tame value of money
    7. Simple payback method
    8. Payback with interest
    9. Present value method
    10. Rate of return
    11. B/C ratio calculation
    12. Internal rate of return

  4. Socio Economic Data[2 hours]
    1. Demographic data
    2. Economic data
    3. Legal data
    4. Social data

  5. Data Management[2 hours]
    1. Collection of data            
    2. Processing of data
    3. Analysis of data
