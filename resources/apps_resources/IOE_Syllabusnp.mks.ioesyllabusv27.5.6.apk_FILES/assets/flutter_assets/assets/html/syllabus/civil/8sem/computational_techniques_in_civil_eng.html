Syllabus

COMPUTATIONAL TECHNIQUES IN CIVIL ENGINEERING

Course Objective:
The knowledge of numerical solutions of various civil engineering problems is essential to the analysis and design of the structures. This course includes the fundamentals of computational techniques required for those problems which cannot be solved by manually and ordinary methods. It teaches the steps required to solve the equations using algorithms and computer programs.

  1. Introduction  (4 hours)
    1. History of numerical computations of civil engineering problems
    2. Brief description of solution techniques
      1. Finite element method
      2. Finite difference method
      3. Boundary element method
      4. Discrete element method
      5. Smoothed particle hydrodynamics
    3. Review of programming methods: (C or FORTRAN or Matlab)
  2. Solutions of linear equations  (6 hours)
    1. System of linear equations
    2. Banded matrices
    3. Data storage and memory optimization
    4. Conjugate gradient method
    5. Fourier Integral
      1. Discrete Fourier Transform
      2. Fast Fourier Transform
  3. Elasticity in solids (6 hours)
    1. Stress displacement relationship
    2. Stress-strain (constitutive) relations
      1. 3D state of solid, Lame constants
      2. Plane stress and plane strain condition
      3. Axi-symmetric stresses
    3. Equilibrium equations
  4. Finite element method (14 hours)
    1. Direct stiffness method
      1. Stiffness matrices for bar, truss and beam element matrices
      2. Transformation matrices for 2D and 3D cases and assembly
      3. Example of a truss
    2. Coordinate system - local, global, natural
    3. Interpolation functions
      1. Pascal triangle
      2. Polynomial function
      3. Lagragian element
      4. Hermite interpolation for beam element
      5. Serendipidity element
    4. Application in solid and frames
      1. Formulation of stiffness matrices for bars, truss, beams and area (triangle) elements
      2. Isoparametric formulation (linear displacement field only) – 2D triangle and quadrilateral
      3. Example of dam: Calculate stresses giving pressure loads using computer programs
      4. Example on wall: Calculate stresses giving vertical loads using computer programs
      5. Example on wall: Calculate stresses giving vertical loads using computer programs
    5. General introduction to pre and post processing
  5. Finite difference method (7 hours)
    1. Finite differences
    2. Explicit scheme and Implicit Scheme
    3. Governing equations of movement of fluid( Momemtum and continuity equations)
    4. Discretization of Kinematic wave Equation (linear and non linear)
    5. Order of accuracy of the scheme and its applications
    6. Numerical diffusion, dispersion and stability of scheme
    7. Applications of the schemes in hydraulic channel routing
    8. Implicit dynamic wave model
    9. Finite difference scheme for Saint-Venant equations
  6. Method of Characteristics (4 hours)               
    1.  Introduction
    2. Characteristics
    3. Initial and boundary conditions
    4. Solution to unsteady flow in pipes
  7. Simulation of Ground water flow (4 hours)
    1. Steady state flow nets and finite difference grid
    2. Simulation of seepage under a dam
    3. One dimensional Implicit Model
    4. Application in river-Groundwater system

Tutorials:

  2.  Solutions of linear equations (6 hrs)
    Theory, definition and concept type questions
    Practical examples, and numerical examples types questions
    Write algorithm and computer programs to solve the problems
  3. Elasticity in solid (2 hrs)
    Theory, definition and concept type questions
    Practical examples of various conditions of stresses
  4. Finite element method (FEM)  (10 hrs)
    Theory, definition and concept type questions
    Practical examples, numerical examples and derivation type questions
    Analyze beams and simple frames    
  5. Finite difference method (4 hours)
    Theory, definition and concept type questions,
    Practical examples, numerical examples and derivation type questions, excercises on Hydraulic channel routing
  6. Method of Characteristics (2 hours)
    7. Theory, definition and concept type questions, Solution of unsteady flow     
  7. Simulation of Ground water flow  (4 hours)
    Theory, definition and concept type questions , excercises on Seepage under dam and River stage –Water table evaluation     
                                                                 
Assignments

  1.  Analyze a 2D dam to find stresses giving water pressure and surcharge
  2. Analyze a 2D wall panel to find stresses giving vertical loads
  3. Analyze 2 storey 2 bay frame and draw bending moments and shear force diagrams
  4. Write source codes to solve Saint-Venant equations marks distribution.
