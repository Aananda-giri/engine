Syllabus

COMPUTER AIDED DRAWING

Course Objective:
To give fundamental knowledge on Computer Aided Drafting (2D and 3D) using common drafting software program.

  1. Introduction(4 hours)
    1. Loading Software, Screen organization
    2. Entering commands: menus, command line, function keys
    3. Commands and System Variables
    4. Coordinate System: entering distances and angles
    5. Starting a new drawing: naming, and saving, ending session

  2. Basic Drawing Commands(12 hours)
    1. Creating point
    2. Creating straight line and construction line
    3. Creating circle, arc and ellipse
    4. Creating polygons
    5. Creating splines

  3. Modifying Commands(8 hours)
    1. Erasing the Object
    2. Creating multiple Objects
    3. Scaling the Object
    4. Creating Chamfer and Fillet
    5. Trimming and Extending of the Object
    6. Breaking and Dividing
    7. Modifying colors, styles, etc the Object

  4. Drawing Aids and Tools (8 hours)
    1. Setting up Units and Limits
    2. Using Ortho, Grids and Snap
    3. Help and Undo Commands
    4. Display Commands
    5. Creating Isometric Drawing

  5. Fine Tuning Drawings and Grouping(4 hours)
    1. Hatching Command
    2. Working with Layers
    3. Creating and Inserting Blocks

  6. Working with Text and Dimensions(4 hours)
    1. Inserting Tex on drawing
    2. Dimension Styles, Dimensioning Commands, Tolerance, Limits,
    3.  Dimension Setup, Dimension Variables, Dimension Scale

  7. Working with Three Dimensional Drawing(12 hours)
    1. Wireframe, Surface and Solid Modeling
    2. Creating 3D Drawing using THICKNESS and ELEVATION Commands
    3. Solid Modeling, Standard
    4. Creating 3D Drawing using EXTRUDE Command
    5. Standard Solid Editing Commands
    6. 3D Modifying Commands: move, rotate, mirror, array
    7. Shading and Rendering and their Options, Motion Path Animations

  8. Plotting Drawings(4 hours)
    1. Layout Management
    2. Plotting 2D and 3D Drawings
    3. Creating multiple views for a 3D drawing

  9. Other Facilities(4 hours)
    1. Use of Script Files
    2. Working with Standard Symbols
    3. Import/Export

Practical:

  1. Familiarization with Software Environment, Setting up Drawing
  2. 2D Drawing Consisting  Straight Lines
  3. 2D Drawing Consisting  Circle and Arc
  4. 2D Drawing Consisting  Ellipse and Polygon
  5. 2D Drawing Using Modifying Commands
  6. Creating Isometric, Creating Hatch, Working with Layers and Blocks
  7. Inserting Text and Dimensions of 2D Drawing
  8. 3D Drawing: Wireframe and Surface Modeling
  9. 3D Drawing:  Solid Modeling
  10. 3D Drawing: Solid Editing and 3D Operations
  11. Plotting 2D and 3D Drawings: Using Script File, Design Center and Import/ Export Facilities
  12. Project 1: Drawing of standard mechanical components: Spring, Nut Bolt, Gear, Cam Profile, etc.
  13. Project 2: Detail Drawing
  14. Project 3: Assembly Drawing

References:

  1. “AutoCAD User's Guide”, Autodesk, 2009.
