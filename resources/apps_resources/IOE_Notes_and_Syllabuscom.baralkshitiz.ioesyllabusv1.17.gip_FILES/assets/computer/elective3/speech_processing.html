SPEECH PROCESSING
CT 785 08

Lecture   : 3
Tutorial   : 1
Practical : 3/2

Course Objectives:

  • To introduce the characteristics of Speech signals and the related time and frequency domain methods for speech analysis and speech compression
  • To introduce the models for speech production
  • To develop time and frequency domain techniques for estimating speech parameters
  • To introduce a predictive technique for speech compression
  • To understand speech recognition, synthesis and speaker identification.
  1. Nature of speech signal
    (8 hours)
    1. Speech production: Mechanism of speech production
    2. Acoustic phonetics 
    3. Digitalmodels for speech signals
    4. Representations of speech waveform
      1. Sampling speechsignals
      2. Basics of quantization
      3. Delta modulation
      4. Differential PCM
  2. Time domain methods for speech processing
    (8 hours)
    1. Time domain parameters of Speech signal
    2. Methods for extracting the parameters
      1. 1Short-time Energy
      2. Average Magnitude
      3. Short-time average Zero crossing Rate
    3. Auditoryperception: psychoacoustics.
    4. Silence Discrimination using ZCR and energy
    5. Short Time Auto Correlation Function
    6. Pitch period estimation using AutoCorrelation Function
  3. Frequency domain method for speech processing
    (10 hours)
    1. Short Time Fourier analysis
      1. Fourier transform and linear filtering interpretations
      2. Sampling rates
    2. Spectrographic displays 
    3. Pitch and formant extraction 
    4. Analysis bySynthesis 
    5. Analysis synthesis systems
      1. Phase vocoder
      2. Channel Vocoder
    6. Homomorphic speech analysis
      1. Cepstral analysis of Speech
      2. Formant and PitchEstimation
      3. Homomorphic Vocoders
  4. Linear predictive analysis of speech
    (10 hours)
    1. Basic Principles of linear predictive analysis
    2. Auto correlation method
    3. Covariance method
    4. Solution of LPC equations
    5. Cholesky method
    6. Durbin’s Recursive algorithm
    7. Application of LPC parameters
      1. Pitch detection using LPC parameters
      2. Formant analysis
      3. VELP
      4. CELP
  5. Application of speech & audio signal processing
    (9 hours)
    1. Algorithms:
      1. Dynamic time warping
      2. K-means clustering and Vector quantization
      3. Gaussian mixture modeling
      4. Hidden Markov modeling
    2. Automatic Speech Recognition
      1. Feature Extraction for ASR
      2. Deterministic sequence recognition
      3. Statistical Sequence
      4. Recognition
      5. Language models
    3. Speaker identification and verification
    4. Voice response system
    5. Speech synthesis
      1. Basics of articulatory
      2. Source-filter
      3. Concatenative synthesis

Practical:

There should be at least 4-6 experiments based on following topics

  1. Spectral analysis
  2. Time-Frequency analysis
  3. Pitch extraction
  4. Formant tracking
  5. Speech enhancement
  6. Audio coding
  7. Speaker recognition

All these lab works may be performed in Matlab or similar softwares capable of processing speech signals. It can also be implemented in hardware if available.
