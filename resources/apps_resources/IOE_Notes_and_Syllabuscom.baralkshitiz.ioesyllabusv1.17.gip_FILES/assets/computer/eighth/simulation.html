SIMULATION AND MODELING
CT 753

Lecture   : 3
Tutorial   : 1
Practical : 3/2

Course objectives:

To provide the knowledge of discrete and continuous system, random numbers generation, queuing system and computer system simulation.

  1. Introduction to Simulation
    (4 hours)
    1. System, model and simulation
    2. Discrete and continuous systems
    3. Model of a system
    4. Types of models
    5. Steps in simulation study
    6. Model development life cycle
    7. Advantage and disadvantage of simulation
    8. Limitations of the simulation techniques
    9. Areas of application
  2. Physical and Mathematical models
    (4 hours)
    1. Static physical model
    2. Dynamic physical model
    3. Static mathematical models
    4. Dynamic mathematical models
  3. Continuous system simulation
    (5 hours)
    1. Differential and partial differential equations
    2. Continuous system models
    3. Analog computer
    4. Analog Methods
    5. Hybrid simulation
    6. Digital-Analog simulators
    7. Continuous System simulation languages(CSSLs)
    8. Feedback systems
  4. Queuing system
    (6 hours)
    1. Elements of queuing system
    2. Characteristics of queuing systems
    3. Model of queuing system
    4. Types of queuing system
    5. Queuing notation
    6. Measurement of system performance
    7. Network of queues
    8. Applications of queuing system
  5. Markov chains
    (3 hours)
    1. Key features of Markov chains
    2. Markov process with example
    3. Application of Markov chain
  6. Random Number
    (10 hours)
    1. Properties of Random Numbers
    2. Generation of Pseudo-Random numbers
    3. Random Number generation methods
    4. Test for random numbers
    5. Generating discrete distribution
    6. Inversion, rejection , composition and Convolution
  7. Verification and validation of simulation models
    (3 hours)
    1. Verification and validation
    2. Verification of simulation models
    3. Calibration and validation of models
  8. Analysis of simulation output
    (4 hours)
    1. Estimation methods
    2. Simulation run statistics
    3. Replication of runs
    4. Elimination of Initial bias
  9. Simulation software
    (3 hours)
    1. Simulation in Java
    2. Simulation in GPSS
    3. Simulation in SSF
    4. Other simulation software
  10. Simulation of computer systems
    (3 hours)
    1. Simulation tools
    2. High Level computer – system simulation
    3. CPU simulation
    4. Memory Simulation

Practicals:

  1. Simulation of continuous system
  2. Simulation of the R-C amplifier circuit
  3. Generation of Random number
  4. Simulation mass spring damper system
  5. Simulation of National econometric system

References:

  2. Geoffrey Gordon, "System Simulation"
  3. A.M. Law and W.D. Kelton, "Simulation and Modeling and analysis"
  4. R. Y. Rubinstein, B. Melamed, "Modern Simulation and Modeling"
  5. S. Shakya, "Lab Manual on Simulation and modeling"
