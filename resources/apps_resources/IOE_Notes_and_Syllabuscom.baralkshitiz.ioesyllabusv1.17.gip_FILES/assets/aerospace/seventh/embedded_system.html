Course Objectives:

This practical, hands-on course introduces the various building blocks and underlying scientific and engineering principles behind embedded systems. The course covers the integrated hardware and software aspects of embedded processor architectures, along with advanced topics such as real-time, resource / device and memory management.

  1. Introduction
    (5 hours)
    1. Course Overview
    2. ARM Architecture
    3. ARM asm Overview
  2. Components
    (6 hours)
    1. Memory Mapped I/O and Buses
    2. Serial Protocols
    3. Timers and Interrupts
  3. ARM
    (6 hours)
    1. ARM Profiling and Optimization
    2. ARM SWI
    3. Sampling, ADCs, DACs
  4. Sensors and Actuators
    (4 hours)
  5. Scheduling
    (8 hours)
    1. Processors
    2. Scheduling and Concurrency
    3. Real-Time Scheduling 1-2
    4. Real-Time Scheduling 2-2
  6. Embedded System Design
    (10 hours)
    1. Design Review
    2. Embedded Linux
    3. Multi-Core and SoC
    4. Embedded Power Management
    5. Embedded Control
  7. Communication System
    (6 hours)
    1. Real-Time Communication
    2. Embedded Wireless Communication
    3. Security for Embedded Systems
