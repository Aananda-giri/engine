Syllabus

GEOGRAPHIC INFORMATION SYSTEM
AR  825 04              

Course objective:

On Completion, a student will be able to

  1. Understand the principle and the use of GIS vocabulary
  2. Carryout basic GIS Operation
  3. Use the knowledge of GIS for settlement study & planning/ urban diversity and infrastructure planning  & inventory assessments
  4. Understand fundamentals of Remote Sensing, Global Positioning System (GPS)
  1. Overview of GIS [6 hours]
    1. Introduction and background of GIS
    2. GIS sub-systems
      1. Data input
      2. Data Storage and retrieval
      3. Data manipulation and analysis
      4. Data output
    3. Components of GIS – Hardware, software, Data, People, Methods
    4. GIS Applications
      1. Different application fields
      2. Key features of GIS
      3. Perspectives on GIS and Sustainability Management
      4. Introduction to Remote Sensing

  2. Map Projection Systems [2 hour]
    1. Map scale
    2. Map projections and its classification
      1. According to geometric surface - azimuthal, cylindrical & conical
      2. According to the property - equivalent, equidistant & conformal
    3. Spatial coordinate system
      1. Geographic coordinate system
      2. Plane coordinate system
    4. Global Positioning System

  3. Data Source and Data Model [4 hours]
    1. Data types
      1. Spatial data
      2. Vector data - Point, Line and Polygon
      3. Raster data – eg. DEM
    2. Comparison between vector and raster data types
    3. Other data types
      1. Image data
      2. Triangular Irregular Network (TIN)
    4. Data sources and geo-referencing of spatial data

  4. Data Analysis [6 hours]
    1. Introduction to Data analysis
    2.  Analysis types
      1. Database Query.
      2. Spatial Query
      3. Overlay Operations
      4. Network analysis.
      5. Buffer Operations

  5. Data Presentation [4 hours]
    1. Common Map Elements
    2. Map Classification
      1. Cadastral Maps
      2. Topographic Maps
      3. Thematic Maps
    3. Bertin’s visual variables
    4. Data type classification
      1. Qualitative data – Nominal
      2. Quantitative data – Ordinal, Interval, Ratio
    5. Map visualization & map presentation techniques
      1. Qualitative data
      2. Quantitative data – Absolute and Relative

  6. GIS Application for Architects and Planners [8 hours]
    1. Urban Land use Planning
      1. Objectives of Landuse Planning – Activities, Location, Implementation
      2. Effective use of GIS for land use planning
    2. Urban Infrastructure Management
      1. Infrastructure types
      2. Principles of GIS in Infrastructure management
    3. Urban Transport
      1. Definition of Urban transport
      2. Aggregate and disaggregate models
      3. Application of GIS in transport planning
    4. Disaster Management
      1. Urban disaster management
      2. Key definitions – Hazard, Risk, Vulnerability
      3. GIS application in disaster management
